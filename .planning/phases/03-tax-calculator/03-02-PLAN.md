    ```typescript
    import type { TaxRules, AgeCategory, TaxRegime, Slab } from '../types';

    export interface HRAInput {
      actualHRA: number;
      basicPlusDa: number;
      rentPaid: number;
      isMetroCity: boolean;
    }

    export interface IncomeTaxInput {
      grossSalary: number;       // Total salary before any deduction
      otherIncome: number;       // Interest, rental, etc.
      fy: string;
      regime: TaxRegime;
      ageCategory: AgeCategory;
      // Old regime only (ignored in new regime):
      deductions?: {
        section80C?: number;
        section80D_self?: number;
        section80D_parents?: number;
        section80CCD1B?: number;
        isSelfSenior?: boolean;
        isParentsSenior?: boolean;
      };
      hra?: HRAInput;            // Only for old regime salaried
    }

    export interface SlabBreakdown {
      slab: string;
      taxableAmount: number;
      tax: number;
    }

    export interface IncomeTaxResult {
      grossIncome: number;
      standardDeduction: number;
      hraExemption: number;
      totalDeductions: number;
      taxableIncome: number;
      slabTax: number;
      slabBreakdown: SlabBreakdown[];
      rebate87A: number;
      marginalRelief: number;
      taxAfterRebate: number;
      cess: number;
      totalTax: number;
      effectiveRate: number;     // totalTax / grossIncome as a percentage
    }
    ```

    **Core functions to implement:**

    1. `computeSlabTax(income: number, slabs: Slab[]): { total: number; breakdown: SlabBreakdown[] }`
       - Iterate slabs; for each slab compute the taxable band = min(remaining, slab.upTo - prevLimit)
       - Accumulate tax and build breakdown entries only for slabs with rate > 0
       - Label each breakdown entry as "Up to ₹X" or "₹X – ₹Y"

    2. `calculateHRAExemption(input: HRAInput): number`
       - Returns min of: actual_HRA, city_pct × (basic+DA), rent_paid − 0.10 × (basic+DA)
       - city_pct: 0.50 for metro, 0.40 for non-metro
       - Returns 0 if rentPaid is 0 (no HRA claim)
       - CRITICAL: base is basic+DA — NOT gross salary

    3. `applyMarginalRelief(slabTax: number, taxableIncome: number, rules: TaxRules['newRegime']): { effectiveTax: number; rebate87A: number; marginalRelief: number }`
       - If taxableIncome <= rebate threshold: rebate = min(slabTax, maxRebate), marginalRelief = 0, effectiveTax = 0
       - If taxableIncome > rebate threshold: rebate = 0; marginalRelief = max(0, slabTax - (taxableIncome - rebateThreshold)); effectiveTax = slabTax - marginalRelief
       - This prevents the ₹12L cliff

    4. `calculateIncomeTax(input: IncomeTaxInput, rules: TaxRules): IncomeTaxResult`
       - Compute standard deduction (from rules for the selected regime)
       - For old regime: compute HRA exemption; cap each deduction at its limit from rules.oldRegime.deductionLimits
       - Compute taxableIncome = grossSalary + otherIncome - standardDeduction - hraExemption - totalOtherDeductions
       - Select slabs: new regime uses rules.newRegime.slabs; old regime uses rules.oldRegime.slabs[ageCategory]
       - Compute slabTax via computeSlabTax()
       - Apply rebate/marginal relief:
         - New regime: use applyMarginalRelief()
         - Old regime: rebate = min(slabTax, rules.oldRegime.rebate87A.maxRebate) if taxableIncome <= threshold; else rebate = 0; no marginal relief
       - cess = (slabTax - rebate87A - marginalRelief) × rules.cess
       - totalTax = taxAfterRebate + cess
       - effectiveRate = (totalTax / (grossSalary + otherIncome)) × 100
       - Return full IncomeTaxResult

    **src/lib/capitalGainsEngine.ts:**

    Exported interfaces:
    ```typescript
    import type { TaxRules, CapitalGainsAssetType } from '../types';

    export interface CapitalGainsInput {
      assetType: CapitalGainsAssetType;      // 'equity' | 'realEstate' | 'other'
      salePrice: number;
      purchasePrice: number;
      holdingMonths: number;
      // Real estate only:
      acquisitionBeforeJuly2024?: boolean;   // true → indexation option available
      indexedCost?: number;                  // User-supplied CII-adjusted cost
    }

    export interface CapitalGainsResult {
      gainType: 'STCG' | 'LTCG';
      rawGain: number;                       // salePrice - purchasePrice
      exemptionApplied: number;              // equity LTCG ₹1.25L exemption
      taxableGain: number;
      taxRate: number | 'slab';              // 'slab' means added to normal income
      taxAmount: number | null;              // null when rate is 'slab' (UI handles it)
      indexationOption?: {                   // Only for pre-July-2024 real estate
        withIndexation: { taxableGain: number; taxAmount: number };
        withoutIndexation: { taxableGain: number; taxAmount: number };
        recommendedOption: 'withIndexation' | 'withoutIndexation';
      };
    }
    ```

    Function `calculateCapitalGains(input: CapitalGainsInput, rules: TaxRules): CapitalGainsResult`:
    - Determine LTCG vs STCG using rules.capitalGains[assetType].ltcg.holdingMonths vs input.holdingMonths
    - Equity LTCG: apply ₹1.25L annual exemption (from rules.capitalGains.equity.ltcg.exemption); taxableGain = max(0, rawGain - exemption); taxAmount = taxableGain × 0.125
    - Equity STCG: taxAmount = rawGain × 0.20 (no exemption, no ₹1.25L offset)
    - Real estate LTCG without indexation: taxAmount = rawGain × 0.125
    - Real estate LTCG with indexation option (acquisitionBeforeJuly2024 === true): compute both branches; set indexationOption; recommendedOption = whichever yields lower taxAmount
    - Real estate STCG / other STCG: rate = 'slab', taxAmount = null (caller adds gain to normal income)
    - Other LTCG: taxAmount = rawGain × 0.125
    - CRITICAL: This engine returns only capital gains tax. 87A rebate is NOT deducted here. Rebate logic lives in taxEngine.ts and applies only against slab tax on normal income.

    **src/lib/gstEngine.ts:**

    Exported interfaces:
    ```typescript
    import type { GstTransactionType } from '../types';

    export interface GSTInput {
      amount: number;
      rate: number;               // e.g. 18 for 18%
      transactionType: GstTransactionType;  // 'intraState' | 'interState'
      amountIncludesGST: boolean; // true = inclusive (breakdown from total); false = exclusive (GST on top)
    }

    export interface GSTResult {
      taxableAmount: number;
      gstAmount: number;
      cgst?: number;              // intraState only
      sgst?: number;              // intraState only
      igst?: number;              // interState only
      totalAmount: number;
      effectiveRate: number;
    }
    ```

    Function `calculateGST(input: GSTInput): GSTResult`:
    - If amountIncludesGST: taxableAmount = amount / (1 + rate/100); gstAmount = amount - taxableAmount
    - If !amountIncludesGST: taxableAmount = amount; gstAmount = amount × rate / 100
    - Intra-state: cgst = sgst = gstAmount / 2; igst = undefined
    - Inter-state: igst = gstAmount; cgst = sgst = undefined
    - totalAmount = taxableAmount + gstAmount
    - effectiveRate = (gstAmount / taxableAmount) × 100
    - Validate rate is in [0, 5, 18, 40, 3, 0.25] — if not, throw Error('Invalid GST rate. Use current rates: 0%, 5%, 18%, 40% (standard) or 3%, 0.25% (special)')
    - NOTE: Do NOT offer 12% or 28% — these slabs were eliminated September 22, 2025
