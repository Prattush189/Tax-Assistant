# Milestones: Tax Assistant

## v0 — Prototype (Pre-planning)

**Shipped:** Before 2026-04-04 (Google AI Studio)
**Phases:** None (built as single prototype)

**What shipped:**
- Chat interface with Gemini AI for Indian tax queries
- Dark/light mode with localStorage persistence
- Basic chart rendering (bar + pie) from AI responses
- Markdown rendering with GFM tables
- Responsive layout with mobile sidebar
- Plugin mode via URL param
- Quick query shortcuts and resource links

**Lessons:**
- Single-file architecture doesn't scale
- API key exposure is a security risk
- Basic visualizations are insufficient for tax data
